# scss-utility

A simple SCSS framework. Includes a Flexbox grid, breakpoints, clearfix, and screen reader classes.

## Installation

$ `yarn add -D scss-utility`

**OR**

$ `npm install -D scss-utility`

**AND** import anywhere SCSS is supported:

`@import '~scss-utility';`
